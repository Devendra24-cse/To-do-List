# Interactive To-Do List

A clean and interactive To-Do List web application built to practice and showcase front-end development skills. It features a custom user interface and convenient "quick-add" functionality for managing daily tasks.

## 📝 Features

- **Quick-Add Buttons:** Pre-set buttons to instantly add common daily routines like "GYM," "Reading," or "Sleep."
- **Custom Input:** A standard text field to type out and add any specific task.
- **Keyboard Support:** Press the `Enter` key to submit a task quickly without needing to click.
- **Task Management:** Remove tasks from your view using the "✕" button once they are completed.
- **Custom UI:** A fully styled interface with warm colors, hover effects, focus outlines, and a hidden scrollbar for a clean look.

## 🛠️ Technologies Used

- **HTML5:** For the structure and layout of the application.
- **CSS3:** For the styling, flexbox alignment, and interactive hover states.
- **JavaScript (Vanilla):** For DOM manipulation, event listeners, and the core application logic.

## 🚀 How to Run

1. Clone or download this repository to your local machine.
2. Open the `index.html` file in any web browser.
3. Start organizing your tasks!
